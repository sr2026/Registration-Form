# Multi-Step Form Wizard

A modern and responsive **Multi-Step Registration Form** built with **React.js**. The application guides users through a step-by-step registration process with validation, reusable components, a progress stepper, and a review page before final submission.

## 🚀 Features

- Multi-Step Registration Form
- Step-by-Step Navigation
- Previous & Next Buttons
- Dynamic Progress Stepper
- Form Validation
- Review Before Submit
- Responsive Design
- Reusable Components
- Toast Notification
- Modern UI Design
- React Icons Integration

## 📋 Form Steps

### Step 1 – Personal Details
- Full Name
- Email Address
- Phone Number

### Step 2 – Address Details
- Address
- City
- State
- Pincode

### Step 3 – Education Details
- College Name
- Degree
- Branch
- CGPA

### Step 4 – Experience Details
- Company Name
- Job Role
- Years of Experience

### Step 5 – Review
- Review all entered information before submission.
